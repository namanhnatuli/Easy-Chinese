# UI Guidelines

## Design style
- clean study-oriented interface
- soft grid/light background
- dark high-focus study card for learning modes
- sidebar navigation
- responsive from mobile to desktop

## Main sections
- Hanzi/Vocabulary
- Grammar
- Lessons / Path
- Review / Progress
- Settings

## Important UI components
- sidebar nav
- top header
- level tabs / filters
- lesson cards
- word data table for admin
- reusable content form
- flashcard panel
- multiple choice panel
- typing panel
- stats cards and charts

## UX rules
- learner actions should be large and keyboard-friendly
- progress should be visible
- anonymous save attempts should prompt sign-in gracefully
- admin forms should support draft / publish later
