# Product Overview

## Vision
Build a Chinese learning web app for Vietnamese learners.

## Primary value
Users should be able to:
- browse words and grammar by level
- study through guided lessons
- practice with multiple learning modes
- save progress if logged in
- review what they have learned

## Target users
1. Vietnamese learners of Chinese
2. Self-study learners preparing for HSK
3. Admin/content creators who manage lessons and vocabulary

## Core learning content
### Word
A word contains:
- simplified chinese
- traditional chinese
- hanzi form if separated in UI/domain
- pinyin
- han viet
- vietnamese meaning
- examples
- HSK level
- topic
- radical
- tags
- optional notes

### Grammar point
A grammar point contains:
- title
- level
- structure
- explanation in Vietnamese
- explanation in English (optional later)
- example sentences
- notes
- tags

### Lesson
A lesson contains:
- title
- slug
- level
- description
- cover / icon (optional later)
- ordered list of words
- ordered list of grammar points
- publication status

## Non-goals for v1
- AI chat tutor
- handwriting recognition
- full gamification
- user-generated courses
